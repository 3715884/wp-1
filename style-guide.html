2. Images and Background Images

Website paragraph text should be mixed with images and the overall look of a website can be enhanced with background images, be they full size images or smaller "tiling" images. Update the links of these sample images once you have uploaded images to your img directory.

3. Fonts

Use webfonts (eg Google Fonts) or the font-face rule to import fonts into your pages rather than using generic fonts such as Arial and Times-New-Roman.

Ideally the headings should be more "fancy" whereas blocks of paragraph text should be more "readable". Headings should ALWAYS be bigger than the main text.

4. Link Styling

Links in the nav, main and footer should be styled differently using the LoVe HAte rule. For example

nav a { 
  /* styles that affect all nav links go here */
}
nav a:link, 
nav a:visited { 
  /* styles that affect just the "non-interacted" nav links go here */
}
nav a:hover, 
nav a:active { 
  /* styles that affect "interacted" nav links go here */
}

There is no :current-page-link pseudo state, but if you look near the top of the page, you will see that there is some PHP code that inserts CSS style to style nav links that have the same name as the current page.
